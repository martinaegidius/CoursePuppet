# Object-detection with Viola-Jones 

## Packages 
This weeks exercises will heavily utilize `opencv`, which you import as: 

```py
import cv2
```
Ensure you have it installed by first activating your virtual conda environment first by running `conda activate course02503` in a terminal and then running 
```pip list```  

in the terminal and check if it is present in the output. 
Else you can install it using 

```pip install opencv-python```



## Advice on how to approach the exercise
The style of this exercise is a bit different than others. 
We advise you to work on every part of the exercise in its own python script, as this allows to maintain an easier overview for you.
If you can't remember how to run Python scripts, check out [our brief guide](../../ex1-setup/running_python_terminal). 
In exercises there are provided some code templates, which you can directly copy into a script to start building your solutions.
