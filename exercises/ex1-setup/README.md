# Installation and setup guide 

The following sections include all you need to get started with this course. Here you'll learn how to install and use _conda_, _VS code_, which you'll use throughout the course. Optionally, you can also learn to setup and use _Git_.

The sections are structured as follows:

- [Conda and VS Code Setup](conda_vscode.md): Guide to setup _conda_ and _VS Code_.

- [Figures in Notebooks](figs_in_nbs.md): Troubleshooting visualizations in _Jupyter Notebooks_.

- [Data and GitHub](data_and_github.md): Outlines the ways of acquiring data for exercise sessions. 

## Exercise data and material (Setting up)

The data and material needed throughout the course can be fetched in multiple ways; see [Data and GitHub](data_and_github.md).

Start by creating an exercise folder to store your data, Python scripts, and notebooks. Download or clone the required data and materials (see [Data and GitHub](data_and_github.md)) and place them in this folder. For example, for Exercise 1 we recommend saving your script or notebook in: `../DTUImageAnalysis/exercises/ex1-IntroductionToImageAnalysis/`.
